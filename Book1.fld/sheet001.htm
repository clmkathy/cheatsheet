VIM cheatsheet
:q close file Combine operators with motions to use them
:qa close all files 2dd delete 2 lines
:x/:wq save and close file cw change one word
c3w change three words
Navigating c4l change four letters
h j k l arrow keys 4x change  4 characters after the cursor
ctrl + u half page up 4p paste 5 times after the cursor
ctrl + d half page down 25,50 lines 25-50
ctrl + B page up :g/^\s*$/d 全局删除空白行
ctrl + F page down :2,7s/^/#/ 把2-7行前面加上#注释
words :g/{pattern}/d delete all lines containing pattern
b / w previous/next word :g!/{pattern}/d delete all lines not containing pattern
ge / e previous/next end of word d$ or D delete to the end of the line
x delete character
Line
0 (数字) start of line exiting insert mode
^ start of line (after whitespace) Esc exit insert mode
$  end of line ctrl+ C eixt insert mode and abort current command
character clipboard
fc go forward to character c x delete character
 Fc go backward to character c clipboard delete line (cut)
yy yank line (copy)
Document p paste
gg First line P paste before
 G last line
 :{number} go to line {number} Find & Replace
{number}G go to line {number} :%s/foo/bar/g replace foo with bar in whole document
{number} j go down {number} lines
{number} k go up {number} lines operaters list
  d delete
Editing y yank
a append c change(delete then insert)
 A append from end of line
i insert > indent right
o next line <  indent left
 O previous line = autoindent
s delete char and insert g~ swap case
 S delete line and insert gU Uppercase
 C delete until end of line and insert gu lowercase
 r replace one character u undo changes
R enter replace mode  Ctrl+R redo changes
